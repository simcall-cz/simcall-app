=========================================
SimCall - n8n Workflow Setup Guide
=========================================

PŘEHLED:
Po každém hovoru ElevenLabs odešle webhook s přepisem.
n8n workflow zpracuje data, pošle přepis do ChatGPT pro analýzu,
uloží nahrávku do Supabase Storage a výsledky do databáze.

=========================================
IMPORT WORKFLOW DO n8n:
=========================================

1. V n8n klikněte na "Workflows" → "Add Workflow" → "Import from File"
2. Vyberte soubor: docs/n8n-workflow-import.json
3. Workflow se naimportuje se všemi uzly

=========================================
NASTAVENÍ CREDENTIALS V n8n:
=========================================

1. Supabase:
   - Credentials → Add → Supabase
   - Host: https://ujvmzylsbblqpqbpbapq.supabase.co
   - Service Role Key: (váš nový service_role key)
   - Po uložení: v každém Supabase uzlu ve workflow vyberte tuto credential

2. OpenAI (ChatGPT):
   - Credentials → Add → OpenAI
   - API Key: váš OpenAI API key

3. ElevenLabs API Key (pro audio download):
   - Credentials → Add → Header Auth
   - Name: xi-api-key
   - Value: váš ElevenLabs API key

=========================================
ODSTRANĚNÍ STARÉHO ÚČTU Z n8n:
=========================================

1. Jděte do Settings → Credentials
2. Najděte starou Supabase credential (se starým URL)
3. Klikněte na ni → Delete
4. Ujistěte se, že žádný workflow nepoužívá starou credential

=========================================
AKTIVACE:
=========================================

1. Ve workflow klikněte na "Active" toggle (vpravo nahoře)
2. Zkopírujte webhook URL (v Webhook uzlu → "Production URL")
3. Nastavte webhook URL v ElevenLabs Dashboard:
   Agent → Webhooks → Post-call webhook URL
4. Aktualizujte N8N_WEBHOOK_URL v .env.local a na Vercelu

=========================================
TESTOVÁNÍ:
=========================================

1. Proveďte testovací hovor v aplikaci
2. Zkontrolujte n8n Executions pro výsledek
3. Ověřte data v Supabase tabulkách (calls, transcripts, feedback)
